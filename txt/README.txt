ai.py : Gère les interactions avec le modèle AI.
calendar_integration.py : Gère l'intégration avec Google Calendar.
commands.py : Extrait et exécute les commandes.
config.py : Contient les configurations globales et les clés API.
main.py : Boucle principale de l'assistant vocal.
stt.py : Gère la reconnaissance vocale.
tts.py : Gère la synthèse vocale.
utils.py : Contient des fonctions utilitaires.
credentials.json : Contient les codes d'autorisations du bot pour l'agenda
token.json : Clé pour éviter de réotoriser à chaque fois, je vous passe celui lié au gmail que je créé pour le bot , si vous voulez le lier à votre agenda vous devrer le supprimer et m'envoyer votre gmail.

Pour lancer le bot : Ouvrez un terminal, rendez-vous dans le repertoire que ce fichier, puis faites :
pip install -r requirements.txt

Puis faites : 
python main.py


Je vous pretes ma clée API de Groq, faites pas n'importe quoi ou créez la votre c'est gratuit.
